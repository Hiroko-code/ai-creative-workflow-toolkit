# Storyboard Template

Use this template to plan a manga page, illustration sequence, short comic, or cinematic beat before generating or drawing final art.

This template intentionally uses generic placeholders. Replace them only with content that is safe to publish.

## Project

- Title: `Project Title`
- Format: `single page manga / four-panel comic / illustration sequence / storyboard`
- Reading direction: `left-to-right / right-to-left / vertical scroll`
- Target mood: `calm / tense / funny / dramatic / mysterious`
- Public-safe summary: `One-sentence summary without private lore`

## Creative Intent

- What should the audience understand?
- What emotion should the page or sequence create?
- What must remain consistent across panels?
- What should be reviewed by a human before generation?

## Characters

| ID | Public-safe role | Visual notes | Emotion | Action |
| --- | --- | --- | --- | --- |
| Character A | protagonist / narrator / visitor | generic clothing or silhouette notes | neutral | enters the scene |
| Character B | companion / rival / observer | generic clothing or silhouette notes | curious | reacts to Character A |

## Location

- Location ID: `Location A`
- Environment: `interior / exterior / fantasy / modern / abstract`
- Time of day: `morning / afternoon / night / unspecified`
- Key props: `Prop A, Prop B`
- Visual constraints: `simple background, clear silhouettes, readable staging`

## Page Or Sequence Layout

| Panel | Shot type | Composition | Character action | Dialogue or caption | Notes |
| --- | --- | --- | --- | --- | --- |
| 1 | wide shot | establish the location | Character A enters | optional caption | keep layout readable |
| 2 | medium shot | focus on interaction | Character B reacts | short line only | avoid crowded text |
| 3 | close-up | show expression | Character A notices detail | none | emphasize emotion |
| 4 | impact shot | reveal or punchline | both characters respond | final short line | preserve story beat |

## Continuity Checklist

- [ ] Character identities are clear.
- [ ] Panel order is understandable.
- [ ] Important props stay consistent.
- [ ] Dialogue is short enough for the layout.
- [ ] The page does not depend on private lore to make sense.
- [ ] No private reference files are named or linked.
- [ ] No generated image paths are included.

## Prompt Draft

```text
Create a clean storyboard based on the layout above.
Preserve panel order, character roles, readable silhouettes, and the intended mood.
Use only the public-safe details in this template.
Do not add private lore, hidden character details, real personal data, or unrelated text.
```
